# Kckstrt
> Even designers can kickstart a project.

Generate a Sinatra app in 4 easy steps:

```
kckstrt generate MyApp
cd my_app
script/setup
foreman start
```

## Features
- [Heroku]-ready
- One-step [installation script][installation]
- Sprockets ([Stylus][] & [Nib][] enabled)
- [Assets helpers][helpers]
